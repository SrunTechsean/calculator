# JavaScript Calculator
A very clean Webpage Calculator with keyboard support made from HTML, CSS, and JavaScript. 
This project was made for [The Odin Project](https://www.theodinproject.com/).

## Feature
- Add, Subtract, Multiply, Divide, Percent
- Keyboard Support (type number and operators)
- Backspace to delete
- Escape to clear

## Quick Start
1. Clone the repository:
```bash
git clone https://github.com/SrunTechsean/calculator.git
cd calculator
```
2. Open index.html in your browser
```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```
